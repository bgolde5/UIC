Contents:

arr.c, arr.h, makefile, main.c


Project Description:

Reads integers from a specified file, 
and stores the values in a dynamically allocated array of
size n, where n is the amount of integers in the specified
file. The contents of the array are then copied, sorted, and
both a linear and binary search are performed.



Instructions:

1. After copying the contents into a directory, 
type the following command in the terminal to compile
this program:

 $make main

2. When prompted, enter the name of the file you wish
to read integers from.

3. Enter a number to search for or -999 to terminate the 
program.
