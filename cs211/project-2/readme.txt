Program Description: This program generates a maze from a text file and attempts to solve the maze. If there is a solution, the path taken is displayed, otherwise a message is printed to the console that the maze has no solution.

 For additional information, see project-2.pdf

Concepts Used: Depth first search, allowing for user debug flags on the command line, stacks using linked lists. 

How to use this programs:

1. Unzip bgolde5proj2.zip
2. In the command line, type: make main
3. In the command line, type: ./maze <filename.txt> 
4. For debugging information, type: ./maze <filename.txt> -d or ./maze -d <filename.txt>
